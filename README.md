注意事项：
1.小车马达驱动板的供电要用H5主板的3v3,GND.（不能与7805共地，取5V电源,控制信号电压要与电源进行匹配！否则无法驱动马达。）
2.目前使用的测速编码器使用5V电源板供电，其输出的信号应该是5V，直接连在H5芯片管脚，作为输入信号，存在信号电压不兼容的隐患。
